# ClashGo - Web3 Brawler & Social Platform

ClashGo is a high-fidelity, highly-strategic **Solana Web3 Auto-Battler & Social Mini-Game Platform** designed to emulate professional B2B Fintech architecture while delivering dopamine-driven 60-second gameplay loops. 

Built with React, Vite, and Lucide Vectors.

## 🌟 Core Features

### 1. The Clash Arena Engine (`Clash Royale Lite`)
- **Real-Time Node Grid**: A reactive interval-ticking auto-battler operating globally at 200ms.
- **Card Deployment System**: Players tap into a regenerating Mana bar (Max 10) to deploy completely distinct SVG-vector characters (Warriors, Whale Tanks, Mages) directly into geometric collision lanes.
- **Opponent AI**: Simulated reactive gameplay dynamically spending mock-mana to generate counter-attacks at scale.

### 2. Web3 Trivia Mode
- A 60-second interactive Trivia engine verifying the player's knowledge of the Solana ecosystem.
- Correct answers yield high-value XP distributions mapped back into the active routing framework.

### 3. Deep Social Strategy (Clan Wars)
- **Persistent Win Streaks**: Multi-tab tracking of global streak patterns globally amplifying SKR yield token outputs.
- **Faction Selection**: Choose between rival forces natively (Solana Degens vs Ethereum Maxis).
- **Global Leaderboard**: A sorting algorithm dynamically mapping your local performance relative to high-tier mock opponents (`0xSolKing`, `0xAlpha`) in an instant UX.
- **Profile Dashboard**: Comprehensive tracking of Arena vs Trivia victory paths.

### 4. B2B Fintech Aesthetics
Instead of aggressive neon/arcade mapping, ClashGo leverages **"Trustworthy Crypto Platform"** architectures:
- Mint Green (`#14F195`) and deep Indigo (`#9945FF`) Orbs rendering dynamically (20-second pulses) behind a technical Navy (`#020617`) grid.
- Crisp Neumorphic 1px borders replacing outdated Glassmorphic blurs.
- Highly legible B2B typography mapped strictly through Google `Inter` & `JetBrains Mono`.

## ⚙️ Installation & Architecture

1. Initialize dependencies using standard PM handlers:
   ```bash
   npm install
   ```
2. Rapidly spin up the Vite caching environment:
   ```bash
   npm run dev
   ```
3. Load securely on localhost:5173 to test global Mock Wallet interaction.

*(Note: Requires a `Node.js` runtime >= 18 for native Vite compilation scripts)*
